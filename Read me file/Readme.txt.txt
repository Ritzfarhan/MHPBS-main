INSTRUCTIONS
------------
1.Download and save the latest server file(for example Xampp and apache 8.0.13.exe) to drive C:
2.Download and save the latest server file which is included inside our github website (MHPBS.RAR) to your exact Drive!
3.Extract the file into your drive
4.Click Start the Apache Button. Indicator changes from red to green when server is running
5.Click Start the MySQL Button. Indicator changes from red to green when server is running
6.Launch PHPMYADMIN then create a new table name "mhpbs" and then extract the "mhpbs.sql" file into the following table
7.Run the website using your localhost server in your computer
